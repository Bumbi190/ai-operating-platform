# Månadspaket — Regler (Content Bible v1)

> Hur Atlas/agenter sätter ihop ett **komplett** månadspaket. Se även `package-structure.md`.

1. **Syfte** [KANON]: leverera ett sammanhängande månadsäventyr (saga + lärande-aktiviteter + firande) kring månadens tema, för familjestund hemma.
2. **Struktur** [KANON]: se `package-structure.md` (10 obligatoriska delar + format-varianter + flöde).
3. **Åldersnivå** [OSÄKER]: förskola–tidig skolålder.
4. **Pedagogiskt mål** [KANON]: temats lärande (se Theme Bible) genom upplevelse + skapande; bygga självförtroende/nyfikenhet; familjegemenskap.
5. **Återkommande mönster** [KANON]: Nova & Pling landar i tema-miljö → utforskar → barnet gör aktiviteter → bockar krysslistan → får diplom → bro till nästa månad.
6. **Tonalitet** [KANON]: varm, magisk, uppmuntrande, trygg; svenska.
7. **Visuella regler** [KANON/OSÄKER]: temats nyckelbild + Nova & Pling kanon-utseende; varumärkespalett (rosa/blå/gult) ovanpå temafärger.
8. **Vad som alltid ska finnas** [KANON]: alla obligatoriska delar i `package-structure.md`.
9. **Vad som aldrig får finnas**: skrämmande/olämpligt; The Prompt-material; nya karaktärer; osäkra barninstruktioner.
10. **Verifierad kanon**: delarna + flödet + diplom/krysslista-mönstret (Känslomånaden + Rymdmånaden).
11. **Osäker kanon**: exakt sidantal/omfång per del; åldersspann.
12. **Kunskapsluckor**: produktions-/tryckspecar; fullständig leverabel-checklista per format (digital/tryck/EPUB).
